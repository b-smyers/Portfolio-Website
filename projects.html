    Team Leader

  • Boop AI Arena

    Repository Owner

    Boop AI Arena is a terminal based extension on the final project for a computer science class. The program was created to allow anyone to create there own AI strategies for the game Boop (a tic tac toe like game), and be able to see how they compare to other AI.

  • Bridge Truss Solver

    Repository Owner

    The bridge truss solver was created using python and matplotlib to analyze and optimize bridge truss designs. The program can handle custom materials and their compressive and tensile strengths.

  • Trebuchet Design Optimizer

    Programmer

    The Trebuchet Design Optimizer programatically and systematically tweaks and analyzes trebuchet designs for maximum distance. The final design was then built, tested, and compared to the theorectical version.
